# Delete Sandboxs

Deleting Sandboxes allows you to manage the cost of long term persistence (coming soon).

## Deleting a Sandbox

```ts
await sdk.sandboxes.delete('sandbox-id');
```

This will:

1. Delete the VM from our infra
2. Soft delete the Sandbox from our database
